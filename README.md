# Curso Perfil da Empresa no Google

Base estratégica criada em 13 de junho de 2026 a partir do conteúdo fornecido pelo produtor.

## Arquivos

- `site/mobile.html`: landing page mobile-first recomendada.
- `site/version-b.html`: conceito visual B para desktop e tablet.
- `site/index.html`: versão inicial da landing page.
- `site/assets/`: retrato, conceitos visuais e vídeo de apresentação.
- `banco-do-curso.csv`: inventário estruturado dos módulos e aulas.
- `diagnostico-editorial.md`: correções e melhorias prioritárias do produto.
- `estrategia-meta-ads.md`: posicionamento, funil, campanhas e métricas.
- `landing-page.md`: estrutura e copy-base da página de vendas.
- `matriz-de-criativos.md`: conceitos, roteiros e textos de anúncios.
- `brief-visual.md`: brief para identidade, landing page e artes.

## Visualização

Abra `site/mobile.html` no navegador para visualizar a versão principal. Todos os CTAs estão conectados ao checkout da Kiwify.

O vídeo `site/assets/boas-vindas.mov` é versionado com Git LFS devido ao tamanho do arquivo.
